# Personal Portfolio Website

Welcome to my personal portfolio! This project showcases my skills, experiences, projects, certifications, blogs, and a daily motivational quote.

## Features

- **About Me:** Introduction and bio.
- **Experience Timeline:** Highlights my professional journey.
- **Education Timeline:** Academic background in a timeline view.
- **Projects, Certifications & Blogs:** Clickable cards linking to detailed pages.
- **Quote of the Day:** Motivational quote fetched from an API.
- **Responsive Design:** Optimized for both desktop and mobile.
- **Modern Tech Stack:** Next.js, TypeScript, Tailwind CSS, Shadcn UI components.

## Tech Stack

- **Frontend/Backend:** Next.js, Tailwind CSS
- **Components:** Shadcn UI
- **State & Data Fetching:** React Hooks, Fetch API
